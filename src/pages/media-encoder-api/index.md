# Media Encoder API

The Media Encoder API is the application layer Media Encoder adds on top of UXP: the encoding queue, presets, jobs, and the scripting hooks you use to automate rendering.

<InlineAlert variant="info" slots="text"/>

UXP for Adobe Media Encoder is in public beta. This reference is being written, and the supported API surface may change before general availability.

Until the full reference lands here, use the [Get Started](../get-started/index.md) section for setup and the development loop, and the [Plugins](../plugins/index.md) section for building panels and scripts.
